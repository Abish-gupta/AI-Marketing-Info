# AI-Marketing-Info

AI-Marketing-Info automates business research with a modern web form and n8n + AI backend. Instantly collect, analyze, and deliver company insights and market profiles.

## Description

Automated business research with n8n and AI: 
- Collect company details, run web and AI analysis, and send results to email, Sheets, or Discord.  
- Review the workflow structure in the screenshot below—JSON and code are private.

## Features

- Modern, icon-enhanced form for client input
- n8n automation workflow for backend data processing
- Firecrawl & Gemini (AI) integration for smart business analytics
- Results delivered via email, Google Sheets, and Discord
